# Open Satellite Observation Capability Index

This project is a cross-satellite observation capability index based solely on publicly available information.

## Scope
- Reference only
- No imagery is hosted, stored, proxied, or distributed
- No proprietary APIs or private systems are accessed

## Purpose
To help users quickly understand which satellite missions may have observed, or may be capable of observing, a given area and time period.

## Disclaimer
All information is indicative and for reference only.
Access to external sites is subject to the policies and availability of the respective data providers.
