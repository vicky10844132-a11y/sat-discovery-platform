What is SAT-DISCOVERY?

SAT-DISCOVERY is a public discovery platform for satellite imagery and geospatial data. We provide tools to search STAC APIs, access open data archives, and discover satellite data portals. Our platform includes interactive mapping with Leaflet, AOI drawing tools, and geospatial analysis capabilities.

What We Do

  • Provide interactive map search with STAC API integration
  • Curate links to public satellite data portals and resources
  • Offer geospatial analysis tools and utilities
  • Help users discover freely available satellite imagery
  • Share practical tools for geospatial data processing

What We Don't Do

  • No data hosting or storage
  • No downloads or file distribution
  • No ordering, tasking, or transactions
  • No commercial partnerships or reselling
  • We do not proxy or cache satellite data

Contact & Contributions

For corrections, additions, or to report issues, please visit our GitHub repository.

All data portal links and resources are publicly available and documented. We welcome contributions to improve the platform.
